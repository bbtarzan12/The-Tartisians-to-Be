# 06. 자율 실행 헌장 (Autonomy Charter)

`goal` 기능으로 6단계 로드맵(`docs/05`)을 끝까지 자율 진행하기 위한 규약. 2026-06-26 확정.

## 1. 목표
플레이 가능한 전문가급 프로토타입 — 시작→플레이→레벨업→5분 생존이 완결되는 한 판.

## 2. 잠긴 결정 (착수 게이트 — 확정됨)

| 항목 | 결정 |
|------|------|
| 커밋 정책 | **검증 통과 마일스톤마다 자동 커밋**(한국어 메시지) |
| 진행 방식 | 마일스톤 완료 **보고 후 자동 진행** |
| 카메라 시점 | **쿼터뷰 45°** |
| 성능 목표 | **60fps @ 동시 적 200체** |
| 카메라 구현 | **Cinemachine** 도입 |
| 이동 방식 | **Rigidbody(키네마틱) 물리 이동** |
| VFX | **VFX Graph** 도입 (M4) |
| 업그레이드 | 8종 기본 구성 (`docs/05` M5), 데이터로 밸런싱 |

## 3. 마일스톤 완료 정의 (Definition of Done)
각 단계는 아래를 **전부** 충족해야 완료 — 하나라도 실패면 다음 단계 진입 금지:
- ✅ 컴파일 에러 0 (`console-get-logs`)
- ✅ 해당 단계 테스트 통과 (`tests-run`)
- ✅ 플레이모드 실측 검증 (MCP 스크린샷/동작)
- ✅ 성능 게이트 (해당 시 — 60fps @200체, `profiler` 확인)
- ✅ 코딩 컨벤션 준수 (`docs/04`)
- ✅ 문서 동기화

## 4. 자율 범위 — 승인 없이 진행
코드 작성/수정, 에셋·프리팹·씬·SO 생성, 합의된 패키지 추가(Cinemachine·VFX Graph), 테스트, MCP 에디터 조작, 문서 갱신, 리팩터링, **검증된 마일스톤 자동 커밋**.

## 5. STOP 조건 — 멈추고 반드시 질문
1. **게임 디자인 분기** — 재미/방향이 갈리는 결정
2. **비가역·파괴적 작업** — 대량 삭제, git history 변경, 외부 발행
3. **반복 실패** — 동일 컴파일/테스트 오류 **3회** 자가수정 실패
4. **합의 외 의존성** — 새 유료/외부 패키지, 아키텍처를 바꾸는 큰 변경
5. **범위 이탈** — 로드맵에 없는 큰 기능

## 6. 검증 루프 (매 작업 후)
```
파일 작성 → assets-refresh → console 에러 확인
   → 에러 시 자가수정(최대 3회) → tests-run → 플레이모드 검증
```

## 7. 실패 처리
자가수정 3회 실패 → 멈추고 원인+로그+선택지 보고. 막힌 지점은 task로 기록.

## 8. 진행 보고
마일스톤 완료마다: **무엇을 했고 / 검증 결과 / 다음 단계**. 중간은 task 리스트로 추적. 사용자는 언제든 인터럽트 가능.

## 9. 품질 가드레일 (타협 없음)
- `docs/04` 컨벤션 강제 — 레거시/C# 10+ 문법 발견 즉시 리팩터
- 모든 스폰 객체 풀링, 핫패스 할당 제로
