# 08. 현재 빌드 기획서 (As-Built)

> 지금 **실제로 구현·동작하는** 게임을 역으로 정리한 문서다. 수치는 코드/에셋의 현재 값이며, 밸런싱·확장 논의의 기준점으로 쓴다. (앞을 내다보는 계획은 `05-roadmap.md`, 이 문서는 "지금 상태")
> 기준 커밋: `dd2a060` · 테스트 51/51 · 200체 @193fps

---

## 1. 한 줄 요약

탑다운 쿼터뷰 3D 뱀파이어 서바이벌류. 플레이어는 WASD로 움직이며 **무기는 자동 발사**, 몰려드는 적을 처치해 **경험치 젬**을 모으고, 레벨업마다 **업그레이드 3택**으로 강해지며 **5분 생존**을 노린다.

## 2. 조작

| 입력 | 동작 |
|------|------|
| WASD / 방향키 / 게임패드 좌스틱 | 플레이어 이동 (XZ 평면) |
| (자동) | 무기 발사 — 사거리 내 최근접 적을 자동 조준 |
| 마우스 클릭 | 레벨업 카드 선택 / 종료 화면 "다시 시작" |

## 3. 게임 플로우

```
시작(Game 씬) → 플레이(자동전투·생존)
   ├─ 적 처치 → XP 젬 드랍 → 자석 흡수 → 경험치
   │     └─ 레벨업 → [일시정지] 업그레이드 3택 → 선택 → 재개 (강해짐)
   ├─ 플레이어 HP 0 → 게임 오버
   └─ 5분 생존 → 생존 성공(승리)
종료 화면 → "다시 시작"(씬 리로드)
```

상태 기계(`GameDirector`): `Playing / LevelUp / GameOver / Victory`. LevelUp·종료 시 `Time.timeScale = 0`.

## 4. 플레이어

| 스탯 | 값 | 비고 |
|------|----|----|
| 최대 체력 | 100 | `PlayerDefinition` |
| 이동 속도 | 6 | 런타임 `RunStats`로 업그레이드 반영 |
| 자석 범위(pickupRadius) | 2.5 | 젬 흡수 반경 |
| 이동 방식 | Rigidbody(키네마틱) MovePosition | XZ 평면, 대각 정규화 |
| 이동 경계 | 아레나 ±19로 clamp | 흐름장(=맵) 밖으로 못 나감 |
| 비주얼 | 캡슐 플레이스홀더(흰색) | 아트 미정 |

## 5. 카메라

Cinemachine 쿼터뷰. 플레이어 추종(Follow) + 항상 바라봄(HardLookAt). 오프셋 `(0, 14, -12)` → 약 45° 내려다보기.

## 6. 적 (3종)

스폰 시 플레이어 주변 반경 18 링에서 등장 → 중앙 시뮬레이션이 추적(seek) + 밀집 회피(separation)로 이동. 접촉 시 플레이어에 초당 데미지.

| 종류 | 체력 | 이동속도 | 접촉 DPS | XP | 크기(scale) | 성격 |
|------|------|---------|---------|----|-----------|----|
| Chaser | 10 | 3.0 | 5 | 1 | 1.0 | 기본 |
| Swift | 6 | 5.0 | 3 | 1 | 0.7 | 빠르고 약함 |
| Brute | 30 | 1.8 | 10 | 3 | 1.5 | 느리고 단단 |

- 비주얼: 빨간 캡슐(공유 머티리얼 → GPU 인스턴싱 배칭)
- AI: `EnemySimulation`이 매 FixedUpdate에 전체 적을 한 번에 갱신(데이터 지향). 이웃 질의는 `SpatialHashGrid`(물리 미사용).
- **군중 겹침(PBD 무푸시):** 힘 기반 separation 대신 **위치 보정**(겹치면 절반씩 밀어 해소). 진행방향 되밀기 없이 → 서로/벽으로 밀어붙이지 않고, 공간 없으면 자연 정지. 벽 근처 흐름은 접선 투영. 검증: 벽에 떼로 몰아넣어도 침투 0.
- **네비게이션(NavMesh 미사용):** `FlowField`(그리드 비용장→방향장, BFS). `FlowFieldController`가 영역(40×40, 셀1 → 40×40=1600칸)·장애물(`NavObstacle`)을 차단 셀로 표시하고 플레이어 셀 변경 시 재계산. 디버그 기즈모(격자·장애물·흐름·비용) 토글 제공.
- **아레나:** 둘레 벽 4개(NavObstacle, ±20)로 40×40 폐쇄 공간 → flow 격자(40×40)와 일치. 바닥 40×40. 플레이어는 ±19 clamp로 항상 격자 안.
- **벽 회피(SDF, 물리 미사용):** 거리장(장애물까지 거리)+그래디언트로 벽 근처에서 부드러운 반발 조향 + 침투 시 정확한 밀어내기 → 적이 벽을 파고들지 않음(런타임 검증: 벽 안 강제 배치 8마리 전원 축출). 적은 자기 셀의 흐름 방향을 O(1) 샘플해 **장애물을 우회**(범위 밖이면 직선 추적 폴백). seek(흐름) + separation 합성.
- 접촉 데미지: 적이 접촉 반경 내면 플레이어에 초당 데미지 적용(과거 `_playerHealth` 미해석 버그 수정됨).

## 7. 스폰 (Wave_Default)

| 항목 | 값 |
|------|----|
| 스폰 간격 | 0.15초 |
| 동시 최대(maxAlive) | 200 |
| 스폰 반경 | 18 |
| 등장 종류 | Chaser / Swift / Brute 무작위 |

> 현재는 단일 웨이브가 일정 간격으로 maxAlive까지 무작위 스폰. 시간대별 난도 상승·보스는 미구현.

## 8. 무기 (MagicBolt · 1종)

| 항목 | 값 |
|------|----|
| 발사 간격 | 0.6초 |
| 데미지 | 5 |
| 투사체 속도 | 14 |
| 관통 | 0 (첫 명중 후 소멸) |
| 사거리 | 12 |
| 투사체 수명 | 2초 |

- 동작: 사거리 내 **최근접 적** 자동 조준, **적 중심 높이에서 발사**(키 작은 적도 명중). **예측 사격**(적 속도 기반 선행 조준, leadFactor 0.6). 투사체는 트리거 충돌로 데미지.
- 비주얼: 노란 구체, 풀링.

## 9. 전투·사망

- 데미지 단일 진입점 `DamageSystem`. 적/플레이어 공용 `Health`(`IDamageable`).
- 적 사망 → **사망 VFX**(VFX Graph `DeathPoof`, 풀링) + `EnemyDiedEvent` 발행 → 젬 스폰.

## 10. 진행 (경험치·레벨·업그레이드)

- **경험치 곡선:** 다음 레벨 요구량 = `5 + (레벨-1)×3` (L1→5, L2→8, L3→11 …). 초과분 이월.
- **XP 젬:** 사망 위치 드랍 → 플레이어 pickupRadius 내면 자석으로 끌려와 흡수.
- **레벨업:** 일시정지 + 업그레이드 **무작위 3택**(중복 없음) → 선택 적용 → 재개. 다중 레벨업은 연속 카드.

### 업그레이드 8종

| 카드 | 효과 |
|------|------|
| 공격력 강화 | 무기 데미지 +3 |
| 강타 | 무기 데미지 +7 |
| 연사 속도 | 발사 간격 ÷1.25 (≈ +25% 연사) |
| 관통 | 관통 +1 |
| 투사체 속도 | +4 |
| 이동 속도 | +1 |
| 최대 체력 | 최대 체력 +25 |
| 자석 범위 | pickupRadius +1 |

> 업그레이드는 런타임 `RunStats`에 누적되어 플레이어/무기 컨트롤러가 즉시 반영.

## 11. HUD (UI Toolkit)

- 상단: **레벨 · 생존 타이머(mm:ss) · 킬 수**
- 체력 바(HP n/max), 경험치 바(XP n/next)
- 레벨업 오버레이: 제목 + 3택 카드 버튼
- 종료 오버레이: "생존 성공!" / "게임 오버" + "다시 시작"

## 12. 승패 조건

- **승리:** 5분(300초) 생존
- **패배:** 플레이어 체력 0

## 13. 기술 스택 (요약)

- Unity 6000.5.1f1 / URP **Deferred+** + GPU Resident Drawer + Occlusion Culling
- New Input System · Cinemachine · VFX Graph · UI Toolkit · Awaitable
- 어셈블리 레이어: Core / Data / Systems / Gameplay / UI / Tests
- 패턴: 타입 EventBus · ServiceLocator · 상태기계 · Object Pooling(적·투사체·젬·VFX 전부)
- **성능:** 동시 적 210체 @ 5.2ms(193fps)
- 네비게이션: 커스텀 **Flow Field**(NavMesh/A* 미사용) — 장애물 우회, O(1)/적
- 자동화 테스트 56개(EditMode 46 + PlayMode 10)

## 14. 현재 한계 / 미구현 (논의 거리)

1. **아트 없음** — 플레이어·적 모두 프리미티브 캡슐. 애니메이션 없음(셰이더 모션/VAT 예정)
2. **무기 1종** — MagicBolt만. 장판·근접·다중 투사체 등 미구현
3. **웨이브 단조** — 시간대별 난도 곡선·보스 없음
4. **카드 UI 기능 위주** — 아이콘·등급·보유현황·리롤 없음(회색 버튼)
5. **최대 체력 업그레이드** — `RunStats`만 갱신, 현재 체력 즉시 회복은 미적용

7. **메타 progression/세이브 없음** (이번 프로토타입 범위 밖)
